SIMPLE RECIPE WEB APPLICATION

Created by Trina Chua

Adding New Recipes

The Add New Recipe section features a clean and intuitive form where users can input the following:

  • Title: The name of the recipe.
  • Ingredients: A field to list all necessary ingredients, allowing for structured input.
  • Instructions: A text area for step-by-step preparation details.
  • Category: Users can classify the recipe into categories like appetizers, desserts, etc.
  • Image Upload: A file upload option to attach an image for better visual representation of the recipe.

Once submitted, the recipe is stored and displayed in the recipe list.


Viewing Recipes

The My Recipes section displays recipes in a card layout, providing:

  • Recipe Preview: Includes a small image, title, and a brief description.
  • Details View: Clicking on a recipe reveals the full details, including:
    • A larger image.
    • Full ingredients list.
    • Step-by-step instructions.
    • Category tags for better organization.

Updating Recipes

The Update Recipe functionality enables users to:

  • Modify any field of an existing recipe, including the title, ingredients, instructions, category, and image.
  • Preview changes in real time before saving updates. This ensures users can easily keep their recipes up to date.

Deleting Recipes

  • A Delete button is available for each recipe, allowing users to remove outdated or unwanted entries.
  • A confirmation step prevents accidental deletions.

Comments Section

Users can interact with recipes by adding comments:

  • Add Comment: A simple text box lets users share their feedback or suggestions.
  • Edit Comment: Users can revise their comments to provide clarity or updated feedback.
  • Delete Comment: Comments can be removed if they are no longer relevant.

Favorites

Users can mark recipes as favorites by clicking a dedicated "Favorite" button.

  • A separate My Favorite Recipes section displays all favorited items, allowing for quick access to frequently used recipes.

Search Bar

A dynamic search feature allows users to find recipes by:

  • Keywords: (e.g., "chocolate").
  • Ingredients: (e.g., "chicken").
  • Categories: (e.g., "desserts").

The search results update in real-time, making it easy to locate specific recipes within a large collection.


Star Rating System

The star rating system enables users to:

  • Rate a recipe from 1 to 5 stars based on their experience.
  • View average ratings to identify popular recipes.
  • Update their ratings to reflect any changes in their experience.

Design and User Experience

  • Background Theme: A cozy kitchen-themed background sets the mood for culinary exploration.
  • Clean Layout: Each section is clearly defined, with sufficient spacing for readability.
  • Responsive Design: The app works seamlessly across devices, from desktops to mobile phones.
