Grocery Tracker App

The Grocery Tracker App is an intuitive and dynamic mobile application built with React Native. It is designed to help users effortlessly manage their grocery shopping lists. With powerful features such as dynamic list management, robust cost tracking, and real-time updates, this app offers a seamless user experience for grocery shoppers.

Project Screens


Key Features of the Grocery Tracker App

  • Add Items: Add new grocery items with details like name, price, quantity, image URL, and category using a simple form.
  • Dynamic Item Display: Items are shown in categories like Fruits, Vegetables, and Cooking, with each item card displaying its name, price, image, and quantity.
  • Search Functionality: A search bar lets you quickly find items by name or category, with real-time updates.
  • Edit Items: Modify existing items on the Edit screen, including name, price, quantity, image URL, and category, with options to save changes or delete the item.
  • Delete Items: A confirmation dialog ensures items are only deleted with user consent.
  • Real-Time Total Cost Calculation: A total cost bar updates dynamically as items are added, removed, or edited.
  • Category-Based Organization: Items are grouped by categories with collapsible headers for easy browsing.
  • User-Friendly Design: Clean interface with clear fonts, vibrant headers, and intuitive buttons.
  • Floating Action Button (FAB): A floating "+" button for quick item addition.
